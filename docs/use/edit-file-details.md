# 编辑文件详情（使用说明）

## 这是什么

“编辑文件详情”用于查看并编辑单个漫画文件的元数据，包括：

- 文件名（重命名）
- 标签（自动保存）
- 书签（bookmarks）

## 重命名怎么用

在“元数据 -> 文件名”处输入新文件名，然后点击“重命名”。

说明：

- 只需要输入“文件名本体”，不需要手动填写扩展名（例如 `.zip`），系统会自动保留原扩展名。
- 重命名会立即作用到磁盘文件与数据库记录（同步执行）。
- 如果目标文件已存在，会提示失败，避免覆盖。

例子：

- 原文件：`"[YuuFreak]Summer Memories[马栏山汉化].zip"`
- 输入：`"Summer Memories（精校）"`
- 结果：`"Summer Memories（精校）.zip"`

## 常见问题

### 1）提示成功但文件没变？

正常情况下不会出现该问题：单文件重命名为同步执行，接口成功即代表磁盘与数据库都已更新。

如果你仍然看到未变化，优先检查：

- 文件是否被其他程序占用（例如正在解压/扫描）
- 是否存在同名文件导致重命名被拒绝

### 2）为什么没有“保存更改”按钮？

该页面改为“按模块自动保存/即时生效”：

- 文件名：点击“重命名”立即生效
- 标签：变更后自动保存；如需按当前标签重命名文件，点击“根据标签重命名文件”
- 书签：添加/删除立即保存

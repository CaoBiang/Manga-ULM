# 布局

## 设计目的

- 统一全站（除阅读器外）的页面边距与内容宽度，减少页面间跳转的视觉割裂。
- 固定左侧侧边栏，避免页面滚动导致导航丢失。
- 只允许右侧内容区域滚动，保证交互一致且更利于阅读。

## 整体结构

```mermaid
flowchart LR
  "Layout" --> "左侧侧边栏（固定）"
  "Layout" --> "右侧主区域"
  "右侧主区域" --> "顶部页头（固定）"
  "右侧主区域" --> "内容区（可滚动）"
```

## 关键规则

- 页面滚动：只滚动右侧内容区，窗口本身不滚动。
- 侧边栏：固定高度 `100vh`，菜单内容过长时在侧边栏内部滚动。
- 页头：位于右侧主区域顶部，滚动内容时保持可见。
